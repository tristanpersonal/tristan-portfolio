    My Works

    Calculator: Description

    This project is designed to help beginners learn how to solve basic integer operations and expressions. It offers an easy-to-follow approach by providing step-by-step explanations of each operation in an expression. Users can enter their expressions, and the tool will calculate and explain the results clearly. Built with HTML for layout, CSS for styling, and JavaScript for functionality, this project aims to make learning math expressions simple, interactive, and engaging for users of all skill levels.

    Notes: Description

    This project demonstrates the developer's skills in creativity, critical thinking, and problem-solving through a CRUD application. Users can create, read, update, and delete records, with data stored in the browser’s localStorage. This application shows how to manage data on the client side without needing a server database. Using HTML for structure, CSS for design, and JavaScript for interactivity, this project highlights the developer’s ability to handle data and design user interfaces effectively.

    WireFrame: Description

    This project showcases the entire process of designing and developing a website, from the initial wireframes to the final product. It starts with creating detailed wireframes in Figma to carefully plan the website’s layout and user experience. These designs are then translated into a working website through careful coding in HTML, CSS, and JavaScript. This project emphasizes the critical importance of thorough planning and thoughtful design in creating a user-friendly and visually attractive website. It effectively demonstrates how both design and development skills come together seamlessly to produce a polished and well-designed final product.

    3Treats: Description

    This project showcases the design and functionality of a responsive website, emphasizing front-end development aspects such as user interface and user experience design. The website performs seamlessly across devices by utilizing media queries for responsiveness. Initial designs were crafted in Figma, and the final product was developed using HTML, CSS, and JavaScript. The project includes an attribution page to credit the owner of the images used. This project demonstrates strong capability in creating a polished, responsive web interface that maintains both aesthetic appeal and functionality across devices.

    Thesis: Description

    The "Artificial Intelligence-Based Manifesto System" is an undergraduate thesis project designed to streamline the process of enlisting passenger information for boats. The system, comprising a computer set and a Logitech camera, employs the Haar Cascade and Local Binary Pattern Histogram algorithms for face detection and recognition. It generates a CSV report of passengers, aiding administrators in efficient data management and supporting search and rescue operations during emergencies.
